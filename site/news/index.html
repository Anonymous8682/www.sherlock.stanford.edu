Introducing a whole new Sherlock on the web#

April XX, 2016#

We're delighted to finally present our new website for Sherlock. Its goal is to centralize documentation, tips, FAQs, news and all information about Sherlock in a single place.

You'll find the following sections to explore:

  • a news page for the latest information and announcements about the cluster: maintenances, new software installations, events...
  • an overview of Sherlock's status and current usage
  • all the cluster documentation
